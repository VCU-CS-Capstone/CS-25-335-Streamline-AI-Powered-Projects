1  Site Overview

  • Primary URL: https://https://paradiseportfolios.glitch.me/ (placeholder).
  • Main navigation: Home · Destinations · Trip Planner · Explore · Blog · Book · Contact Us

2  Individual Pages

2.1  Home (index.html)

  • This is a welcome page that immediately hits the user with queries directly integrated with the Trip Planner chatbot.
  • Background fades in and out of snapshots of beaches to get users in the mood to travel. This format persists across the site.
  • Recommended Blog Post section injects a random article on each load via client‑side JavaScript.

2.2  Destinations (destination.html)

  • Organised by region (Caribbean, Mexico, Hawaii, etc.).
  • Each card embeds a 30‑45 s AI‑generated video (<video> with .no-fullscreen class) plus a short description.
  • Top search bar allows the user to search for videos in real time (title text match).

2.3  Trip Planner (trip-planner.html)

  • Embeds the conversational TravelBot powered by an OpenAI fine‑tuned model.
  • Auto‑sends the visitor’s query from the Home page search via the ?message= URL param.

2.4  Blog Overview (blog.html)

  • Grid of destination guides. Typing in the search bar filters by title keywords.
  • Every card links to a detailed blogs/<slug>.html page that ends with a single CTA button: Plan Your Adventure Now! (Sandals affiliate link).

2.5  Book (book.html)

  • Links users to Sandal's booking website with Amanda's referral code for affiliate marketing.
  • Contains a promotional video for code 777 at checkout.

2.6  Contact Us / About

  • Static informational pages describing social links and ways to reach out to Paradise Portfolios.

3  AI Chatbot (TravelBot)

  • Launch by visiting the Trip Planner page.
  • Trained on a curated list of partner destinations. When asked about an unsupported place it suggests the closest match and stays on‑topic with travel.
  • Conversation history persists for the session; refreshing clears the log.
Tip: Type “family trip under $5 k” or “best honeymoon beach October” for personalised ideas and direct booking links.

4  AI Videos & Interactive Maps

  • All destination videos are generated with MiniMax using Sandals/Beaches images and narrated scripts.
  • Vertical & horizontal formats are available for reuse on social channels via an AI video generator on the backend.
  • The Leaflet map auto‑clusters pins; click a pin to jump to the corresponding blog post

5  Dark / Light Mode

  • Toggle via the sun/moon icon in the navbar. The preference is stored in localStorage.theme.
  • System default is light; the site remembers the last choice for 30 days.

6  Responsive Behaviour & Accessibility

  • Navbar collapses to a hamburger menu below 768 px.
  • CTA buttons expand full‑width on mobile for easier tapping.
  • Videos keep a 16∶9 ratio using aspect-ratio CSS.
  • Colour palette meets WCAG AA contrast; alt text is provided for all images.
