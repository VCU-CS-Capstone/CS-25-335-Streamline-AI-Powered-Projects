Paradise Portfolios Deployment Guide

Option 1: Access via Source Code (Local Deployment)

Step 1 – Install prerequisites

  • VS Code or Cursor
  • Python

Step 2 – Install PIP

# Open a new terminal in VS Code / Cursor
pip install pip

Step 3 – Open the project

  1. Clone the repository:

    gh repo clone VCU-CS-Capstone/CS-25-335-Streamline-AI-Powered-Projects

  2. In VS Code choose File ▸ Open Folder… and select the cloned folder.

  3. Enter the project backend and install Node:

    cd chatbot-backend
pip install node   # Installs Node.js wrapper for Python

  4. Start the server and resolve missing packages:

    node server.js
# Read each error message and install the missing package, e.g.
pip install <package-name>

Step 4 – Personalize the project

  1. Add videos

    Download every video (except the 2.20.25 folder) from
    Google Drive collection
    and place them in assets/videos.

  2. Set your OpenAI API key

    • Create a key in OpenAI dashboard.
    • Paste it into the project .env file.

  3. Finetune the model

    1. Go to OpenAI Finetune and click Create.

    2. Choose gpt-4o-mini.

    3. Upload chatbot_data.jsonl.

    4. After training finishes, update chatbot-backend/server.js:

      model: 'ft:gpt-4o-mini-YYYY-MM-DD:personal:YOUR-MODEL-NAME:ID'

Option 2: Access via Glitch (Global Deployment)

Step 1 – Create a Glitch account

  • Glitch

Step 2 – Remix the project

Visit the project page and click Remix:
paradiseportfolios on Glitch

Step 3 – Personalize the project

Follow the same personalization steps as in Option 1:

  • Add your OpenAI API key to the .env file.
  • Finetune the model with the dataset and update chatbot-backend/server.js.
